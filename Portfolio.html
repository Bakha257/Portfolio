Мои проекты

Ниже представлены некоторые из моих проектов. Нажмите на контейнер, чтобы перейти к проекту на GitHub.

Flight Booking

Кроссплатформенное приложение для поиска и бронирования авиабилетов. Используется Amadeus API для получения актуальных рейсов, Stripe — для оплаты, а PDF-генерация — для создания билетов. Веб-версия на React, мобильная — на Expo (React Native).

Room Reservation System

Лёгкое и простое приложение для бронирования комнат без фреймворков. Реализован выбор дат, валидация и отправка заявки. Чистый интерфейс, адаптивная верстка и интерактивность с помощью JavaScript.

Student Tracking

Веб-приложение, позволяющее преподавателям и администраторам отслеживать посещаемость и успеваемость студентов. Система включает как frontend (HTML, CSS, JavaScript), так и backend (Node.js + MySQL) части. Через веб-интерфейс можно добавлять студентов, отмечать посещения и просматривать статистику.

Computer Systems Page

Информационный сайт о кафедре ИТ в университете Turiba. Включает разделы о преподавателях, проектах студентов и контактной информации. Адаптивный, удобный сайт, подходящий как для абитуриентов, так и для студентов.
